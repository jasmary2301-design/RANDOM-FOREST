# RANDOM-FOREST
Random Forest dataset contains student-related attributes such as IQ, Previous Semseter Result, CGPA, Academic Performance, Internship Experience, Extra Curricular Score, Communication Skills each record represents an individual student and their final placements status.

This project implements Random Forest to classify to predict whether a college student will be placed or not placed based on academic performance,skills,and extracurricular factors.Random Forest is an ensemble learning algorithm that improves prediction accuracy by combining multiple decision tress. The model achieved hign accuracy, demonstrating the effectiveness of Random Forest for educational and career outcome prediction.
